保存場所--------------
ローカル環境にフォルダを保存します。
例（c:windows\user\--\map\networkx
注意：onedriveに保存しないでください。

仮想環境--------------
anaconda promptを起動して、'conda create --name map_env'と入力します。
"map_env"という仮想環境が構築されます。
'conda activate map_env'と入力します。

実効方法--------------
anaconda promptを実行し、以下のコードを実行してapp.pyがあるディレクトリに移動します。
'cd C:windows\user\--\map\networkx'

初回だけ以下のコードを記述して必要なライブラリをインストールします。
'pip install -r requirements.txt'

'python app.py'を記述してローカルホストとpythonを起動します。
"Debugger PIN: ---"が表示されるまで待機する。
web ブラウザでlocalhost:5000を入力する。

操作方法---------------
対象領域に移動する。
shift + ドラッグ　で表示領域選択。
南西からドラッグでネットワークデータ抽出範囲を選択し、「ネットワークデータ抽出」をクリックする。
ダブルクリックで中心地点を選択します。
スタートをクリックするとネットワーク分析を開始します。
shp出力でlocal環境のshpファイルにshpフォルダを作成します。


注意事項---------------
javascriptを使用していますが、特別な環境構築は必要ありません。
ダブルクリックが使いづらいかもしれません。
ホイールクリックで地図をドラッグできるようにしましたが、機能が不安定です。
